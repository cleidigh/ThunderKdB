Features

  • Reply++ button/message context menu/keyboard: shift-alt-G:

    • Start a new email compose. Use 'to', 'cc, 'bcc' of the currently selected email. Clean away subject, body, attachments, so that it is a new email.

  • Alt/Reply++ button/Alt+message context menu:

    • Just as standard button use, but keep old subject.

  • Shift/Reply++ button/Shift+message context menu:

    • Add your identity as bcc, so that you automatically force a copy to yourself.

  • Other addons:

    • Nostalgy++: easy archiving/moving of emails to their destination folders, with autocomplete of folder names and rules for suggested folder. Compose window: always display cc or bcc, easy selecting input fields (to, cc, bcc, body, attachments), etc.

    Text insert++: Easy inserting standard text. br+ctrl-D converts to best regards. Define your own replacements. Cursor can be in the middle of the replacement text.

    Attachment image viewer: Download dialog shows thumbnail of image. Image viewer with zoom, rotate, flip, slideshow etc.

    Xnote++: Add notes to your emails to remind or to plan a reply.
