СИНТАКСИС СОБСТВЕННЫХ ЗАГОЛОВКОВ

%%1 = автор
%%2 = получатели 
%%3 = получатели (CC)
%%4 = тема
%%5 = дата
%%6 = автор (только имя) 
%%7 = автор (только адрес)
%%8 = получатели (только имя) 
%%9 = получатели (CC - только имя) 

Текст между {{ и }} будет показан только если заголовок существует
К примеру

To : %%2{{ и %%3}} 

будет выглядеть как

To : mail1@xxx.org (если CC отсутствует)
To : mail1@xxx.org и mail2@xxx.org (если CC - mail2@xxx.org)

СИНТАКСИС ДЛЯ СОБСТВЕННОГО ФОРМАТА ДАТЫ

%d = день месяца, 2е цифры ("01" - "31")
%D = день недели, текст на английском, 3 буквы ("Fri")
%m = месяц ("01" - "12")
%M = месяц, текст на английском, 3 буквы ("Jan")
%y = год, 2е цифры ("99")
%Y = год, 4е цифры ("1999")
%i = минуты ("00" - "59")
%s = секунды ("00" - "59")
%H = час, 24ти часовой формат ("00" - "23")
%h = час, 12ти часовой формат ("01" - "12")
%A = "AM" или "PM"
%a = "am" или "pm"
%e = space-filled date of month
%z = timezone offset from UTC
